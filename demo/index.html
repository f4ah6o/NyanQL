NyanQL + HTMX quick demo

Send a message, receive HTML from NyanQL, and render it below.

What this demo shows

  • SQL templates: api.json maps /demo_message to two SQL files.
  • HTML response: HTMX sends HX-Request, so NyanQL renders templates/demo_message.html.
  • SQLite storage: messages are persisted to stamps.db.
  • BasicAuth: this form uses the default neko:nyan credentials.
Try JSON response
curl -u neko:nyan -X POST http://localhost:8443/demo_message -d "message=hello"
